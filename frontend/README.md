# TLO — Frontend (Vite + React)

Interfaz del **sistema operativo TLO** (Transportes Ligeros de Occidente): dashboard, viajes, liquidaciones, catálogos, usuarios y marca.

## Desarrollo

```bash
npm install
npm run dev
```

Por defecto Vite usa el puerto **8080**. Configura `VITE_API_BASE_URL` (por ejemplo `http://localhost:4000/api/v1`) en `.env` para conectar al backend.

## Build

```bash
npm run build
npm run preview
```
